Project Overview

Title: Improving Factual Reliability in Agent-Based Literature Surveys


Domain: Healthcare Literature


Framework: OpenManus


Presented by: Prajwal Jaiswal


Project Advisor: Prof. Haibo Yang

1. Motivation
Trust and Reliability in AI Agents
Large Language Model (LLM)-based agents are increasingly used for knowledge synthesis in fields such as healthcare, finance, law, and academic research.

A significant limitation is that agent outputs often sound confident but may be factually incorrect, which limits real-world deployment and trust.

Problems in Literature Survey Agents
Agents generating multi-step summaries across multiple papers are prone to:

Unsupported or hallucinated claims.

Incorrect attribution to sources.

Overgeneralization of limited evidence.

These errors are often subtle and difficult to detect automatically.

Focus on Healthcare Literature
Healthcare has a very low tolerance for factual errors and hallucinations.

Medical literature contains complex, nuanced claims, including statistical results and distinctions between correlation and causation.
+1

If verification methods work on healthcare data, they are likely to generalize to other domains.

2. Proposed Solution
System Architecture
The project proposes adding a Verification Layer to the OpenManus agent pipeline.
+1


Modular Component: Operates after agent reasoning and document retrieval.

Core Functions:

Analyzes generated claims.

Evaluates retrieved evidence.

Checks claim-evidence alignment.

Flags unsupported or inconsistent claims before the final output is delivered.

System Workflow

User Query: Research Question/Topic.
+1


OpenManus Agent: Retrieves scientific papers and generates/synthesizes claims.


Verification Layer: Performs claim analysis, evidence matching, and fact-checking.


Final Output: A verified literature summary that is either verified or flagged for issues.

4. Evaluation Plan
The project will compare baseline OpenManus agents with verification-enhanced agents using identical queries and document sets.


Key Metrics:


Claim Support Rate: Percentage of claims supported by retrieved evidence.


Citation Accuracy: Whether cited papers actually support the attached claims.


Evidence Match Quality: Alignment strength between claims and retrieved passages.


Hallucination Rate: Percentage of claims not grounded in any document.


Human Evaluation: Manual review using a rubric.
